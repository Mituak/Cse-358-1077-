# Description:
This program draws one triangle using OpenGL. At first, the triangle is cyan in color. Then it slowly changes its color between cyan and magenta using animation. The window title shows my full ID. I also added keyboard controls to change the triangle color and behavior.

# Language & Tools:

1. C++
2. OpenGL (GLFW + GLAD)
3. VS Code
4. Windows OS

# How to Compile and Run:

1. Open the terminal in the project folder.
2. Compile the program.
3. Run the executable file.

# Controls:
Press ‘W’ → Triangle becomes white while holding the key.
Release ‘W’ → Animation (cyan ↔ magenta) starts again.
Press ‘R’ → Triangle becomes red permanently and animation stops.
Press ‘M’ → Closes the window.

# Output:

1. One triangle.
2. Cyan to magenta color animation.
3. White and red color changes using keyboard.
